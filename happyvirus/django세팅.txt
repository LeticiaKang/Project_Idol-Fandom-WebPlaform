가상환경 폴더 만들기
    ▶python -m venv myvenv

가상환경 진입하기
    ▶f1
    ▶select Interpreter → 만든 가상환경 선택

새로운 터미널 생성
    ▶ctrl+shift+`

장고설치하기
    ▶python -m pip install django

WebProject 폴더 생성하기
    ▶mkdir HappyVirus

프로젝트 생성
    ▶django-admin startproject config . 

프로젝트 변동사항을 데이터베이스에 적용
    ▷migration 생성
        ▶python manage.py makemifrations
    ▷DB 변동사항 적용
        ▶python manage.py migrate

서버 실행
    ▶C:\HappyVirus>python manage.py runserver 

언어와 시간을 한국 값으로 바꾸기
    ▶settings.py파일
        LANGUAGE_CODE = 'ko-kr'
        TIME_ZONE = 'Asia/Seoul'
    

앱 생성(←프로젝트에 기능을 추가하기 위해서는 앱을 생성해야 한다.)
    ▶django-admin startapp myapp

서버 실행 후 브라우저에 http://localhost:8000/myapp 요청
    ▶404오류 발생
    ▶webproject/urls.py 파일에 요청한 myapp/ URL에 해당되는 매핑이 없어서 발생

오류해결하기 위해서 webproject/urls.py 파일에 myapp/ URL 추가하기
    ▶webproject/urls.py 파일 수정
        from django.contrib import admin
        from django.urls import path

        ★from myapp import views

        urlpatterns = [
            path('admin/', admin.site.urls),
            ★path('myapp/', views.index),
        ]

URL 매핑에 추가한 뷰 함수 views.index 생성
    ▶myapp/views.py 파일에 index 함수 추가
        from django.http import HttpResponse
        
        def index(request):
            return HttpResponse("안녕하세요 happyvirus에 오신것을 환영합니다.")
